  • Player-specific interactions.

  • Methods: CollectItem(), UseItem(), Interact(), SolvePuzzle(), etc.

  1. EnemyController Script (inherits from Character)

  • Control enemy behaviors.

  • Methods: Patrol(), ChasePlayer(), Attack(), etc.

  1. AIController Script

  1. DifficultyController Script

  • Adapt game difficulty.

  1. Inventory Script

  • Manage player inventory.

  1. CombatController Script

  • Manage combat interactions.

  • Methods: Attack(), Defend(), Dodge(), etc.

  1. PuzzleController Script
